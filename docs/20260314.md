### 5. Tarea 0.4: Schema de Base de Datos

**Agente Invocado:** @sql-specialist  
**Archivo Creado:** `migrations/001_init.sql`

**Tablas Creadas:**
1. **jobs** - Trabajos de transcripción
   - Campos: id, file_path, file_hash, file_size_mb, status, queue
   - Timestamps: created_at, started_at, completed_at
   - Tracking: retry_count, progress, chunks_total, chunks_completed
   - Usuario: user_id, platform
   - Error: error_message

2. **transcription_cache** - Cache de transcripciones
   - Campos: file_hash (PK), transcription, access_count
   - Timestamps: created_at, last_accessed

3. **rate_limits** - Control de límites por usuario
   - PK compuesta: (user_id, platform, window_start)
   - Campos: file_count

4. **metrics** - Métricas del sistema
   - Campos: id, metric_name, metric_value, recorded_at

**Índices Creados:**
- `idx_jobs_status` - Búsquedas por estado
- `idx_jobs_queue` - Filtrado por cola
- `idx_jobs_user` - Rate limiting
- `idx_cache_hash` - Acceso a cache
- `idx_jobs_platform` - Consultas por plataforma
- `idx_jobs_created_at` - Consultas temporales
- `idx_metrics_name` - Acceso por métrica
- `idx_metrics_timestamp` - Consultas temporales

**Validación:**
- Script SQL ejecutado exitosamente en SQLite
- 4 tablas creadas correctamente
- 8 índices funcionando
